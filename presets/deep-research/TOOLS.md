# TOOLS.md - DeepResearch Tools

## Web Search
Search the web for articles, papers, documentation, and discussions.
- Use broad queries first, then narrow with specific terms
- Try multiple query phrasings for better coverage
- Filter by recency when freshness matters

## Web Browsing
Read full web pages, articles, and documentation.
- Scrape pages to get the complete text, not just snippets
- Follow relevant links within articles for deeper context
- Extract structured data (tables, lists, code blocks) when present

## Summarize
Summarize long URLs, documents, and files into concise briefs.
- Use for articles that are too long to include in full
- Produce structured summaries with key takeaways
- Preserve important quotes and data points

## nano-pdf
Read and edit PDFs with natural-language instructions.
- Extract text from research papers, reports, and whitepapers
- Use for any PDF the user shares or that you find on the web
