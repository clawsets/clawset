# @clawset/competitor-radar

Competitive intelligence agent that tracks competitors, surfaces new features, and generates backlog ideas.

## Skills

- `github` — GitHub integration for backlog management
- `slack` — Slack integration for team notifications

## Schedule

Runs on weekdays at 9 AM (`0 9 * * 1-5`).

## Install

```bash
clawset install competitor-radar
```
