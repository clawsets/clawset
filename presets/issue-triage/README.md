# @clawset/issue-triage

Autonomous project manager that triages issues, assigns tasks, and generates standups.

## Skills

- `github` — GitHub integration for issue and PR management
- `slack` — Slack integration for team notifications

## Schedule

Runs on weekdays at 9 AM (`0 9 * * 1-5`).

## Install

```bash
clawset install issue-triage
```
